## Status Legend

- Done: usable implementation exists and has focused tests or clear runtime wiring.
- Partial: real implementation exists, but important production/product behavior is still missing.
- Scaffold: package/service/app exists and boots, but only has placeholder or health-check behavior.
- Planned: not implemented yet.
